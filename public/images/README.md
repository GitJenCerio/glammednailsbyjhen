- Format: JPG or WebP
- Hero image: 1920x1080px
- Service/About images: 800x600px minimum
- Gallery images: 1200x1200px (square) or similar
- Keep file sizes under 500KB for optimal performance
- Use Next.js Image optimization for automatic compression
